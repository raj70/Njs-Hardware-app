# Njs-Hardware-app
Hardware Store Sample App

Uses JavaScript and html at the front end.
Uses JavaScript and MongoDb at the back end.

### Uses following libs:
* express
* Webpack (s)
* babel (s)
* polyfill
* cors
* sass
* html-webpack-plugin
* axios

* mongoose
* multer
* underscore.string
* bcrypt

* jsonwebtoken
